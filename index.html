Your First Week Deploying llm-d in Production

Every item below addresses a specific failure mode from the incidents on this page. Do these before your first real traffic.

Deploy model cache DaemonSetPre-pull weights to node-local NVMe before your first rollout. Without this, 24 pods pulling 140GB simultaneously = 38-minute outage.
Configure compound readiness probesCheck /health AND /v1/models AND warmup inference. Default probe returns 200 before model is loaded.
Set HPA stabilization to 15+ minutesDefault 5-minute window causes oscillation. Model load takes 4-10 min. You will thrash between 8-20 pods.
Set Topology Manager to single-numa-nodeOn every GPU node. NUMA misplacement doubles decode latency and is invisible to standard monitoring.
Add GPU fragmentation monitoringTrack largest_free_block / total_free. Alert at 0.3. nvidia-smi total free is not enough.
Monitor finish_reason distributionAlert when finish_reason=length exceeds 5%. Silent truncation looks like "the model got dumber."
Configure PodDisruptionBudgetsSet minAvailable to 75%. Without PDBs, scale-down events terminate pods mid-inference.
Set KV cache eviction to frequency-weightedPure LRU evicts your hottest system prompts first under load. That is the exact wrong order.
Add RDMA health checks to readiness probesIf running disaggregated serving: TCP health is not RDMA health. The RDMA fabric will fail silently.
Deploy priority queues by context lengthA single 128K request costs 256x more compute than a 500-token request. Same queue = guaranteed starvation.
Namespace cache keys by tenant IDHash collisions in shared prefix caches cause correctness bugs that look like hallucinations.
Schedule pod recycling every 24 hoursGPU memory fragments over time. After 18 hours of mixed workloads, OOM kills start with 40% free memory.
Set maxSurge=1, maxUnavailable=0Slow rollouts are correct rollouts. The alternative is a 38-minute outage.
Treat ConfigMap updates as rolling updatesEventually consistent propagation breaks session affinity for 30+ seconds. Add config version to health checks.
